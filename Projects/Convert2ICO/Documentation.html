Convert2ICO - Button Documentation

Add to Context Menu: Adds the "Convert To > ICO" entry to your Windows right-click context menu for supported image files (.png, .jpg, .jpeg, .bmp, .gif).

Remove ICO Entry: Removes only the "ICO" child entry under "Convert To" without affecting other entries you may have added under the same node.

Remove 'Convert To' Node: Removes the parent "Convert To" node completely. Use this if you want to clean up all related entries.
